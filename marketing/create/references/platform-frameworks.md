## LinkedIn

**Optimal length**: 1,300–2,000 characters  
**Best timing**: Weekdays 7–9am local time  
**Critical window**: First 60–90 min after posting = algorithm decides reach

### Post Structure
```
[Hook — 2-5 words, no question]
[blank line]
[Tension — why does this matter? 1-2 sentences]
[blank line]
[Development — the real content, 3-6 short paragraphs]
[blank line]
[Validation — proof, number, example]
[blank line]
[Conclusion — question or soft CTA, never a perfect aphorism]
```

### Hook Patterns
- Ultra-short statement: "I almost quit." / "Nobody told me this."
- Number lead: "18 months ago, I had 0 followers."
- Observation: "Something weird happened."
- Contrarian opener: "Onboarding is overrated."

### Algorithm Rules
- Reply to every comment in first 90 min (author replies = strong signal)
- No external links in post body (kills reach) — put links in comments
- Native video > external video > images > text
- Polls boost reach but attract low-quality engagement

### Post Formats
| Format | Best for | Length |
|--------|----------|--------|
| Story | Personal journey, failure, lesson | 1,500–2,000 chars |
| Framework | Systematic insight, how-to | 1,300–1,800 chars |
| Observation | Hot take, trend, data point | 800–1,300 chars |
| Contrarian | Challenge common wisdom | 1,000–1,600 chars |

## X/Twitter

**Single tweet**: 240-280 chars (leave 10% buffer)  
**Thread**: 5-12 tweets optimal  
**Best timing**: Weekdays 8–10am, 6–8pm

### Tweet Structure (single)
```
[Hook — first 5 words scroll-stop]
[Core idea — be specific]
[Optional: stat, example, or question]
```

### Thread Structure
```
Tweet 1: [Hook/claim — strongest statement]
Tweet 2: [Context — why this matters]
Tweets 3–N: [Evidence, examples, steps]
Tweet N-1: [Counter-intuitive insight]
Tweet N: [Conclusion + CTA or open question]
```

### Algorithm Rules
- Quote RT > RT for reach
- First reply from yourself (add a thread link, extra context)
- Images boost engagement ~35%
- Don't delete and repost — kills the tweet
- Long-form (Notes) still low reach, use sparingly

### Tweet Patterns
- "Hot take: [thing people believe] is [actually wrong/different]"
- "I tested [thing] for [time period]. Here's what happened:"
- "[Number] things I wish I knew before [experience]:"
- "Unpopular opinion: [opinion]"

## Email

**Subject line length**: 30–50 characters  
**Body length**: 150–300 words for nurture; up to 600 for launch  
**Best timing**: Tuesday–Thursday, 9–11am

### Email Structure
```
[Subject line — curiosity gap OR personal OR benefit]
[Preview text — extends subject, not repeat]

[1-line hook — immediate, specific]

[Problem — 2-3 sentences max]

[Solution/idea — 2-4 sentences]

[3 bullets MAX if needed — only for scannable lists]

[Single CTA — one link, one action]

[Sign-off]

[P.S. — second most-read element, use for bonus value or secondary CTA]
```

### Subject Line Formulas
| Type | Formula | Example |
|------|---------|---------|
| Curiosity gap | "[Thing] is broken" | "Your onboarding is broken" |
| Personal | "I was wrong about [thing]" | "I was wrong about virality" |
| Benefit | "How to [result] in [time]" | "How to get 100 leads in a week" |
| Urgency | "[Event] ends [time]" | "Last day for early access" |
| Social proof | "[Number] people did [thing]" | "312 founders tried this" |
| Question | "Are you making this mistake?" | — |

### Sequence Types
| Type | Length | Goal |
|------|--------|------|
| Welcome | 3–5 emails | Onboard + build trust |
| Nurture | Ongoing, weekly | Stay top of mind |
| Launch | 5–7 emails | Drive action |
| Re-engagement | 3 emails | Win back cold subs |

## Ad Copy

### Facebook/Instagram
**Hook**: Must stop scroll in 3 seconds  
**Structure**: Problem → Solution → Proof → CTA  
**Length**: 125 chars primary text, 40 chars headline

```
[Primary text — conversational, UGC-style]
[Headline — benefit or curiosity]
[Description — clarify or add urgency]
[CTA button — matches headline action]
```

**UGC-style patterns:**
- "I spent [time] trying to [goal]. Then I found [solution]."
- "Why does [problem] still exist in [year]? Because nobody does [solution]."
- "[Number] [target audience] switched to [solution] this month. Here's why."

### Google Ads
**Headlines**: 30 chars × 3 (rotated)  
**Descriptions**: 90 chars × 2  
**Match headline to landing page H1**

**Headline formula slots:**
1. Benefit headline: "[Do thing] [X]% faster"
2. Feature headline: "[Tool/product] for [audience]"
3. Social proof: "Join [number]+ [audience type]"
4. Action headline: "Start [goal] today"
5. Emotional headline: "Stop [painful thing]"

## Landing Pages

**Above the fold**: User must understand the offer in 5 seconds  
**Section count**: 7–9 sections optimal  
**CTA**: Primary CTA appears in hero + at least 2 more sections

### Above-the-Fold Formula
```
[Headline — 6-10 words, specific benefit or transformation]
[Subheadline — who it's for, what they get, how fast — 1-2 sentences]
[Primary CTA button — action verb + outcome]
[Social proof strip — logos, numbers, testimonial fragment]
```

### Section Flow
```
1. Hero        → Hook + primary CTA
2. Problem     → "Sound familiar?" — 3 specific pain points
3. Solution    → How your thing works (mechanism, not features)
4. Proof       → Social proof (testimonials, case studies, logos)
5. Features    → 3-6 key features with benefits
6. Pricing     → Simple, anchored, clear value
7. FAQ         → Objections as questions (5-7)
8. Final CTA   → Repeat hero offer with urgency
```

### Headline Patterns
| Pattern | Formula | Example |
|---------|---------|---------|
| Benefit+Speed | "[Do X] in [time]" | "Launch your store in 24 hours" |
| Pain Killer | "Stop [pain]" | "Stop losing customers at checkout" |
| Identity | "For [specific person] who [specific situation]" | — |
| Transformation | "From [before] to [after]" | "From waitlist to 1,000 users" |
| Question | "Tired of [specific pain]?" | — |
| Simple Promise | "[Specific result] or [guarantee]" | — |
